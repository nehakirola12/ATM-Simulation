# 🏧 ATM Simulation (Python Project)

## 📌 Overview

This is a simple Python-based ATM Simulation project that allows users to perform basic banking operations through a command-line interface. It is designed for beginners to understand core programming concepts like functions, loops, and condition handling.

---

## 🚀 Features

* 💰 **Display Balance** – Check current account balance
* 💵 **Deposit Money** – Add money to your account
* 💸 **Withdraw Money** – Withdraw money with validation
* 📄 **Transaction Statement** – View transaction history
* 🔁 **Menu-driven system** – Easy navigation with options

---

## 🛠️ Technologies Used

* Python (Basic concepts)

  * Functions
  * Loops
  * Conditional statements
  * Lists

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the program:

```
python atm.py
```

---

## 🧠 How It Works

* The program starts with a default balance.
* Users interact with a menu to choose actions.
* Each transaction updates the balance and is stored in a list.
* The statement option displays all previous transactions.
